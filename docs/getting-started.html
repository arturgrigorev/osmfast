Installation

OSMFast is a pure Python tool with zero dependencies. It works with Python 3.7+ on Windows, macOS, and Linux.

# Clone or download the repository
git clone https://github.com/osmfast/osmfast.git
cd osmfast

# Verify installation
python osmfast.py --help

You should see a list of available commands. That's it - no pip install, no virtual environments, no dependencies to manage.

✓ Pure Python 3 standard library
✓ No external dependencies
✓ Cross-platform compatible
✓ ~7,000+ features/second

Getting OSM Data

OSMFast works with .osm XML files. Here's how to get map data for your area of interest:

Option A: OpenStreetMap Export (Small Areas)

  1. Go to openstreetmap.org/export
  2. Navigate to your area of interest
  3. Click "Export" to download the .osm file

Option B: Geofabrik (Cities, Regions, Countries)

  1. Visit download.geofabrik.de
  2. Navigate to your continent → country → region
  3. Download the .osm.bz2 file and extract it

Option C: Overpass API (Custom Queries)

# Use Overpass Turbo: https://overpass-turbo.eu/
Tip
For this tutorial, we'll assume you have a file called map.osm in your working directory.

Extracting Features

OSMFast provides specialized commands to extract specific feature types. All outputs support GeoJSON, JSON, and CSV formats.

Extract Buildings

# Extract all buildings to GeoJSON
python osmfast.py buildings map.osm -o buildings.geojson

# Filter by height (for 3D visualization)
python osmfast.py buildings --min-height 10 map.osm -o tall_buildings.geojson

Extract Roads

# All roads
python osmfast.py roads map.osm -o roads.geojson

# Only major roads
python osmfast.py roads --type primary,secondary map.osm -o major_roads.geojson

Extract Points of Interest

# All POIs
python osmfast.py poi map.osm -o pois.geojson

# Restaurants and cafes
python osmfast.py food map.osm -o restaurants.geojson

# Shops
python osmfast.py shop map.osm -o shops.geojson

Generic Extraction

# Extract with custom filters
python osmfast.py extract --accept-nodes amenity=hospital map.osm -o hospitals.json

# Extract to CSV for spreadsheet analysis
python osmfast.py extract --format csv --accept-nodes shop=* map.osm -o shops.csv
Available Extractors
buildings, roads, poi, amenity, shop, food, healthcare, education, tourism, water, landuse, trees, parking, transit, railway, power, and more.

Filtering Data

Use the filter command to create subsets of OSM data. This is useful for reducing file size or preparing data for specific analysis.

Filter by Tag

# Keep only highways
python osmfast.py filter --accept-ways highway=* map.osm -o roads_only.osm

# Keep restaurants
python osmfast.py filter --accept-nodes amenity=restaurant map.osm -o restaurants.osm

# Multiple filters (OR logic)
python osmfast.py filter --accept-ways highway=* --accept-ways railway=* map.osm -o transport.osm

Filter by Bounding Box

# Extract a specific geographic area
# Format: --bbox min_lat min_lon max_lat max_lon
python osmfast.py filter --bbox 49.45 11.05 49.48 11.09 map.osm -o downtown.osm

Combine Filters

# Roads in a specific area
python osmfast.py filter \
  --bbox 49.45 11.05 49.48 11.09 \
  --accept-ways highway=* \
  map.osm -o downtown_roads.osm
Note
The filter command outputs OSM XML format. Use extract for GeoJSON/CSV output.

Rendering Maps

OSMFast can render your OSM data as static images, PDFs, interactive web maps, or 3D visualizations.

PNG Image

# Basic PNG render
python osmfast.py render map.osm -o map.png

# Custom size and style
python osmfast.py render --width 1920 --height 1080 --style dark map.osm -o map_dark.png

Available Styles

--style default
--style dark
--style light
--style blueprint

PDF for Publication

# Publication-ready PDF (no Type 3 fonts)
python osmfast.py render --format pdf map.osm -o map.pdf

# A4 size for papers
python osmfast.py render --format pdf --page-size a4 --orientation landscape map.osm -o figure.pdf

Interactive HTML (Leaflet)

# Interactive web map
python osmfast.py render --format leaflet map.osm -o map.html

# With satellite tiles
python osmfast.py render --format leaflet --tiles esri-satellite map.osm -o satellite.html

3D WebGL Visualization

# 3D buildings with Three.js
python osmfast.py render --format webgl map.osm -o city3d.html

# Dark style 3D
python osmfast.py render --format webgl --style dark map.osm -o city3d_dark.html

Routing & Analysis

OSMFast includes routing and network analysis capabilities built on the road network in your OSM data.

Find a Route

# Route between two points (lat,lon format)
python osmfast.py route --from 49.45,11.08 --to 49.46,11.09 map.osm

# Walking route
python osmfast.py route --from 49.45,11.08 --to 49.46,11.09 --mode walk map.osm

# Save route as GeoJSON
python osmfast.py route --from 49.45,11.08 --to 49.46,11.09 -o route.geojson map.osm

Multi-Stop Routes

# Route through multiple waypoints
python osmfast.py route-multi 49.45,11.08 49.46,11.09 49.47,11.10 map.osm

Distance Matrix

# Calculate distances between multiple points
python osmfast.py distance-matrix points.csv map.osm -o matrix.json

Isochrones (Travel Time Areas)

# Areas reachable within 10 minutes
python osmfast.py isochrone --from 49.45,11.08 --minutes 10 map.osm -o isochrone.geojson

Network Analysis

# Export road network as graph
python osmfast.py network map.osm -o network.json

# Find network bottlenecks
python osmfast.py bottleneck map.osm -o bottlenecks.geojson

# Calculate centrality
python osmfast.py centrality map.osm -o centrality.geojson

Python API

OSMFast can also be used as a Python library:

from osm_core import OSMFast, OSMFilter

# Create filter
osm_filter = OSMFilter()
osm_filter.add_accept_filter('nodes', 'amenity', 'restaurant')
osm_filter.set_bounding_box(49.45, 11.05, 49.48, 11.09)

# Extract features
extractor = OSMFast(osm_filter)
result = extractor.extract_features('map.osm')

# Access results
restaurants = result['features']['amenities']
print(f"Found {len(restaurants)} restaurants")
