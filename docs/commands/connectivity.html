osmfast connectivity

Analyze road network connectivity

← Back to Documentation
Description

The connectivity command analyzes how well connected the road network is. It identifies: - Number of connected components - Size of largest component - Isolated road segments - Network coverage statistics Useful for identifying disconnected areas or data quality issues.

Usage
osmfast connectivity map.osm
osmfast connectivity map.osm --mode drive
osmfast connectivity map.osm --mode walk -f geojson -o components.geojson
Examples
Basic analysis
osmfast connectivity city.osm
Walking network
osmfast connectivity city.osm --mode walk
Cycling network
osmfast connectivity city.osm --mode bike
Export components
osmfast connectivity city.osm -f geojson -o components.geojson
Show all components
osmfast connectivity city.osm --show-components
Options
Option Description
-m, --mode Travel mode: drive, walk, bike (default: drive)
-o, --output Output file (default: stdout)
-f, --format Output format: text, json, geojson
--show-components Output all components as GeoJSON
Output

Connectivity analysis showing: - Total nodes and edges - Number of connected components - Largest component size and coverage % - List of isolated segments (if requested) - GeoJSON of components (if geojson format)
