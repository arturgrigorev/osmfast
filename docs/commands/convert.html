osmfast convert

Convert between file formats

← Back to Documentation
Description

The convert command transforms OSM files to various formats. Supports: - GeoJSON with proper geometry - Plain JSON structure - CSV with flattened tags - OSM XML Optionally calculates area and length for features.

Usage
osmfast convert map.osm -o map.geojson
osmfast convert map.osm -o map.csv --flatten-tags
osmfast convert map.osm -o map.json --include-area --include-length
Examples
To GeoJSON
osmfast convert city.osm -o city.geojson
To CSV
osmfast convert city.osm -o city.csv
With area/length
osmfast convert city.osm -o city.geojson --include-area --include-length
Flattened CSV
osmfast convert city.osm -o city.csv --flatten-tags
Ways only
osmfast convert city.osm -o roads.geojson --ways-only
Tagged only
osmfast convert city.osm -o features.geojson --tagged-only
Compact JSON
osmfast convert city.osm -o city.json --compact
Options
Option Description
-f, --format Output format: geojson, json, csv, osm (auto-detect from extension)
--nodes-only Only convert nodes
--ways-only Only convert ways
--tagged-only Only elements with tags
--include-area Calculate area for polygons
--include-length Calculate length for ways
--flatten-tags Flatten tags into columns (CSV)
--compact Compact output (no indentation)
-o, --output Output file (required)
Output

Converted file in specified format: - GeoJSON: Point/LineString/Polygon geometries - JSON: Nodes and ways with coordinates - CSV: Tabular data with optional flattened tags - OSM: Standard OSM XML format
