osmfast network

Export road network as graph for analysis

← Back to Documentation
Description

The network command exports the OSM road network as a graph structure suitable for network analysis, routing algorithms, or visualization in graph tools. The graph includes: - Nodes at intersections and road endpoints - Edges representing road segments - Edge weights for distance and travel time - Road attributes on edges

Usage
osmfast network map.osm -o network.json
osmfast network map.osm -o network.graphml --format graphml
osmfast network --mode walk map.osm -o pedestrian.json
osmfast network --stats map.osm
Examples
JSON network
osmfast network city.osm -o network.json
GraphML for Gephi
osmfast network city.osm -o network.graphml --format graphml
Walking network
osmfast network --mode walk city.osm -o walk.json
With speeds
osmfast network --include-speeds city.osm -o network.json
Options
Option Description
-o, --output Output file
-f, --format Output format: json, graphml, geojson, csv
--mode Travel mode filter: drive, walk, bike, all
--directed Create directed graph (respects oneway)
--include-speeds Include speed estimates and travel times
--stats Show network statistics only
Output

JSON: {nodes: [...], edges: [...]} structure GraphML: Standard graph format for Gephi, NetworkX, etc. GeoJSON: Edges as LineStrings with properties CSV: Edge list with attributes
