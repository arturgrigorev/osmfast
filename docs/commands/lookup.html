osmfast lookup

Find features at or near coordinates

← Back to Documentation
Description

The lookup command performs reverse geocoding - finding what features exist at or near given coordinates. It: - Searches nodes and ways within radius - Calculates distance to each feature - Returns sorted results (nearest first) Useful for identifying locations on a map.

Usage
osmfast lookup map.osm --lat 51.5 --lon -0.1
osmfast lookup map.osm --lat 51.5 --lon -0.1 --radius 100m
osmfast lookup map.osm --lat 51.5 --lon -0.1 -o features.geojson
Examples
Basic lookup
osmfast lookup city.osm --lat 51.5 --lon -0.1
Wider radius
osmfast lookup city.osm --lat 51.5 --lon -0.1 --radius 500m
More results
osmfast lookup city.osm --lat 51.5 --lon -0.1 --limit 50
JSON output
osmfast lookup city.osm --lat 51.5 --lon -0.1 --json
GeoJSON file
osmfast lookup city.osm --lat 51.5 --lon -0.1 -o nearby.geojson
Nodes only
osmfast lookup city.osm --lat 51.5 --lon -0.1 --type nodes
Options
Option Description
--lat Query latitude (required)
--lon Query longitude (required)
-r, --radius Search radius (default: 50m, e.g., 100m, 1km)
-n, --limit Maximum results (default: 10)
-t, --type Element type: nodes, ways, all (default: all)
--json Output as JSON
-o, --output Output file (GeoJSON)
Output

Feature list sorted by distance: - Feature type (node/way) - Name and description - Distance in meters - Coordinates - All tags
