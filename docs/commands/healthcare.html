osmfast healthcare

Extract health facilities

← Back to Documentation
Description

The healthcare command extracts hospitals, clinics, pharmacies, and medical practices from OSM data. Categories include: - Hospital: hospitals, clinics - Doctor: general practitioners, doctors - Specialist: dentists, optometrists, physiotherapists - Pharmacy: pharmacies, chemists - Emergency: emergency departments, urgent care Includes accessibility and specialty information.

Usage
osmfast healthcare map.osm -o healthcare.geojson
osmfast healthcare map.osm --category hospital -o hospitals.geojson
osmfast healthcare map.osm --emergency-only -o emergency.geojson
Examples
All healthcare
osmfast healthcare city.osm -o healthcare.geojson
Hospitals only
osmfast healthcare city.osm --category hospital -o hospitals.geojson
Pharmacies
osmfast healthcare city.osm --type pharmacy -o pharmacies.geojson
Emergency services
osmfast healthcare city.osm --emergency-only -o emergency.geojson
Dentists
osmfast healthcare city.osm --type dentist -o dentists.geojson
Statistics
osmfast healthcare city.osm --stats
Options
Option Description
-t, --type Filter by type: hospital, pharmacy, dentist, etc.
-c, --category Filter by category: hospital, doctor, specialist, pharmacy, emergency, other
-f, --format Output format: geojson, json, csv (default: geojson)
--stats Show statistics only
--emergency-only Only facilities with emergency services
-o, --output Output file (default: stdout)
Output

Healthcare facilities with: - healthcare_type: Facility type - name: Facility name - operator: Operating organization - specialty: Medical specialties - emergency: Has emergency services - wheelchair: Accessibility - phone, website: Contact info
