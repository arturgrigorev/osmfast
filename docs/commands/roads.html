osmfast roads

Extract road network with attributes and geometry calculations

← Back to Documentation
Description

The roads command extracts the road network from an OSM file with comprehensive attributes: - Road classification (primary, secondary, residential, etc.) - Road name and reference numbers - Speed limits (tagged or estimated from road type) - Surface type and smoothness - Lane count and lane-kilometers - One-way restrictions - Length, sinuosity, and bearing - Infrastructure flags (sidewalk, lighting, bridges, tunnels) Supports filtering by road hierarchy levels and infrastructure type.

Usage
osmfast roads map.osm -o roads.geojson
osmfast roads --class major map.osm -o highways.geojson
osmfast roads --named-only map.osm -o named_roads.geojson
osmfast roads --stats map.osm
Examples
All roads
osmfast roads city.osm -o roads.geojson
Major roads only
osmfast roads --class major city.osm -o highways.geojson
Primary roads
osmfast roads --type primary city.osm -o primary.geojson
Named roads
osmfast roads --named-only city.osm -o named.geojson
Road statistics
osmfast roads --stats city.osm
Options
Option Description
-o, --output Output file (default: stdout)
-f, --format Output format: geojson, json, csv, shapefile
--class Road class: major, minor, path, all
--type Specific highway type (e.g., primary, residential)
--min-length Minimum segment length in meters
--named-only Only roads with names
--stats Show statistics only, no export
Output

GeoJSON/Shapefile with properties: - id: OSM way ID - highway: Road type - name: Road name - ref: Reference number (e.g., A1, M25) - maxspeed: Speed limit - lanes: Number of lanes - surface: Road surface type - oneway: One-way restriction - length_m: Length in meters - length_km: Length in kilometers - sinuosity: Curvature ratio (1.0 = straight) - bearing: Direction in degrees - speed_kph: Effective speed - travel_min: Travel time in minutes - lane_km: Lane-kilometers - has_sidewalk: Sidewalk present - is_lit: Street lighting - bridge: Bridge type - tunnel: Tunnel type
