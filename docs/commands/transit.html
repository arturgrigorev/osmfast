Documentation / Feature Extraction / transit

osmfast transit

Extract public transport data

← Back to Documentation
Description

The transit command extracts public transport stops and stations from OSM data. It identifies: - Bus stops and bus stations - Train stations and halts - Tram stops - Subway/metro entrances - Ferry terminals Includes operator, network, and accessibility information.

Usage
osmfast transit map.osm -o transit.geojson
osmfast transit map.osm --type bus -o bus_stops.geojson
osmfast transit map.osm --stats
Examples
All transit
osmfast transit city.osm -o transit.geojson
Bus stops
osmfast transit city.osm --type bus -o bus_stops.geojson
Rail stations
osmfast transit city.osm --type rail -o stations.geojson
Statistics
osmfast transit city.osm --stats
Subway entrances
osmfast transit city.osm --type subway -o subway.geojson
CSV format
osmfast transit city.osm -f csv -o transit.csv
Options
Option Description
-f, --format Output format: geojson, json, csv (default: geojson)
--type Transit type: bus, rail, tram, subway, ferry, all (default: all)
--stops-only Only extract stops (no routes)
--stats Show statistics only
-o, --output Output file (default: stdout)
Output

Transit stops/stations with: - transit_type: Mode of transport - feature_type: stop, station, platform, etc. - name: Stop/station name - ref: Reference number - operator: Transport operator - network: Transit network - routes: Route references - wheelchair: Accessibility
