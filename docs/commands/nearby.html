osmfast nearby

Find features within radius of other features

← Back to Documentation
Description

The nearby command finds target features within a specified radius of source features. This is useful for: - Finding pharmacies near hospitals - Locating parking near shops - Analyzing service coverage - Proximity-based analysis

Usage
osmfast nearby map.osm --source amenity=hospital --target amenity=pharmacy --radius 500m
osmfast nearby map.osm --source amenity=school --target shop=* --radius 1km
osmfast nearby map.osm --source tourism=hotel --target amenity=restaurant --radius 200m -o nearby.geojson
Examples
Pharmacies near hospitals
osmfast nearby city.osm --source amenity=hospital --target amenity=pharmacy --radius 500m
Shops near schools
osmfast nearby city.osm --source amenity=school --target shop=* --radius 1km
Restaurants near hotels
osmfast nearby city.osm --source tourism=hotel --target amenity=restaurant --radius 200m
Statistics only
osmfast nearby city.osm --source amenity=bank --target amenity=atm --radius 100m --stats
Export to GeoJSON
osmfast nearby city.osm --source amenity=hospital --target amenity=pharmacy --radius 500m -o nearby.geojson
Options
Option Description
--source Source feature filter (e.g., amenity=hospital)
--target Target feature filter (e.g., amenity=pharmacy)
-r, --radius Search radius (e.g., 500m, 1km)
-o, --output Output file (default: stdout)
-f, --format Output format: geojson, json, csv
--stats Show statistics only
Output

For each source feature: - Source feature info - List of nearby target features - Distance to each target - Count of nearby targets GeoJSON includes both source and target geometries.
